Submitting Assignments with GradeBridge

What You Will Do

Submitting an assignment through GradeBridge takes four steps:

  1. Download the assignment file from Canvas.
  2. Open the GradeBridge app and enter your answers.
  3. Download your completed solution files.
  4. Upload your solution to Gradescope (and Canvas if instructed).

Each step is explained below. Read through the whole guide once before you start so nothing comes as a surprise.

Before You Begin

You will use three tools:

Before You Begin
Tool What it is
Canvas Your course website — where you pick up the assignment file.
GradeBridge Student Submission App A web app at https://veriqai.github.io/GradeBridge-Student-Submission/ — where you enter and export your answers.
Gradescope Where you submit your completed solution files for grading.

Note on file locations: The app works the same on any computer (Windows, Mac, or Linux). On Windows, downloaded files will appear in your Downloads folder (C:\Users\YourName\Downloads). On a Mac they appear in ~/Downloads.

Before you open the app — get organized first.

Read through your assignment carefully and identify everything you will need to submit (written answers, screenshots, images, etc.). Collect all of these materials into a single folder on your computer before you open GradeBridge. For example, Documents\EEC1\Lab1_Submission\. Your assignment file will list exactly what is required.

Having everything in one place before you start will make the process smooth and uninterrupted.

Step 1 — Download the Assignment File from Canvas

  1. Log in to Canvas and go to the assignment you want to complete.
  2. Download the assignment file. It will be a JSON file. The filename will either be named or at least end in assignment_spec.json.
  3. The file will save to your Downloads folder. Leave it there for now; you will need it in the next step.

Step 2 — Open GradeBridge and Enter Your Answers

  1. Open a web browser and go to the GradeBridge Student Submission App:
    https://veriqai.github.io/GradeBridge-Student-Submission/

  2. Enter your full name (exactly as it appears in Canvas) in the left panel.

  3. Click Upload JSON and select the assignment file you downloaded in Step 1.

  4. Work through the assignment: type your answers into the text boxes provided and upload any required screenshots or images into the image fields.

Math equations: You can type mathematical expressions using LaTeX notation. Wrap inline math in $...$ and display math in $$...$$. A live preview appears below the text box as you type. Click Math Equation Format Help in the sidebar for examples.

AI-graded questions are marked with a purple "AI Graded" badge. A word count is displayed below your answer as a guide to how much detail is expected — it is not enforced and will not prevent you from submitting. Focus on answering the question thoroughly rather than hitting a specific word count.

  1. Your work saves automatically. The app saves your progress to your browser every few seconds — you will see a green "Saved" timestamp at the bottom of the left panel. You can safely close the browser and return later without losing your work.

Switching computers or browsers? The auto-save only works in the same browser on the same computer. If you need to continue your work elsewhere, click Save Backup in the left panel. This downloads a backup JSON file you can reload on any computer or browser. To restore: return to the app, re-enter your name, then click Load Work and select your backup file.

Step 3 — Download Your Completed Solution Files

  1. When you have finished entering all your answers, the bottom bar shows:
Step 3 — Download Your Completed Solution Files
Button What it does
Save Backup Saves a backup of your work-in-progress (not for submission)
Download for Gradescope (blue) Downloads your submission file and PDF — submit these to Gradescope
Preview Shows a formatted view of your submission before downloading

  1. Click Preview if you want to review your submission first. Scroll through to check that you have answered every question. Click Back to return and make corrections.

  2. Click the blue Download for Gradescope button. A single ZIP file will download named:
    {YourName}_{CourseCode}_submission.zip
    For example: Jane_Doe_EEC1_submission.zip
    This is the file you will submit to Gradescope.

    The ZIP contains both your submission data and a PDF copy of your answers — you do not need to open it or extract anything. Just upload the ZIP as-is.

The file will appear in your Downloads folder.

Step 4 — Submit Your Solution on Gradescope

  1. Log in to Gradescope and navigate to the assignment.

  2. Upload the .zip file you downloaded in Step 11.

    Important: Make sure you are uploading the ZIP file ending in _submission.zip, not the original assignment file you downloaded from Canvas in Step 1, and not a backup file. The files have similar names — double-check before you submit.

  3. If your instructor also requires a Canvas submission, upload the same _submission.zip file there as well.

Troubleshooting

My work disappeared when I reopened the browser.
Auto-save stores your work in the browser you used. If you opened a different browser or cleared your browser data, the auto-saved session will not be there. Always use Save Backup if you plan to close the browser, switch computers, or use a different browser.

The app says "Invalid assignment file."
You may have uploaded a backup file instead of the assignment file, or a file from a different course. Re-download the assignment JSON from Canvas and try again.

I accidentally uploaded the wrong assignment.
Click Upload JSON again in the left panel and select the correct file. Your answers will be cleared when a new assignment loads.

The math preview is not showing.
Check that you have closed your $` delimiters correctly — every opening `$ needs a matching closing $. Refer to Math Equation Format Help in the sidebar for correct syntax.

I submitted the wrong file to Gradescope.
Contact your instructor as soon as possible. Gradescope allows resubmission before the deadline in most cases.

I took photos with my iPhone and the file is in HEIC format.
HEIC is now supported natively — just upload the file as-is. The app will automatically convert it to JPEG before saving. If for any reason the conversion fails (you will see an error message), send the photo to your computer via AirDrop or Mail — iOS converts to JPEG automatically during sharing — and upload that file instead.
