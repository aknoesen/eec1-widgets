Submitting Assignments with GradeBridge

What You Will Do

Submitting an assignment through GradeBridge takes four steps:

  1. Download the assignment file from Canvas.
  2. Open the GradeBridge app and enter your answers.
  3. Download your completed solution files.
  4. Upload your solution to Canvas.

Each step is explained below. Read through the whole guide once before you start so nothing comes as a surprise.

Before You Begin

You will use two tools:

Before You Begin
Tool What it is
Canvas Your course website — where you pick up and turn in assignments.
GradeBridge Student Submission App A web app at https://veriqai.github.io/GradeBridge-Student-Submission/ — where you enter and export your answers.

Note on file locations: The app works the same on any computer (Windows, Mac, or Linux). On Windows, downloaded files will appear in your Downloads folder (C:\Users\YourName\Downloads). On a Mac they appear in ~/Downloads. This guide uses Windows paths in the examples.

Before you open the app — get organized first.

Read through your assignment carefully and identify everything you will need to submit (written answers, screenshots, data files, images, etc.). Collect all of these materials into a single folder on your computer before you open GradeBridge. For example, Documents\EEC1\Lab1_Submission\. Your assignment file will list exactly what is required.

Once you are in the app, you will not be able to pause and go looking for files without risking losing your progress. Having everything in one place before you start will make the process smooth and uninterrupted. You will also save your output files to this same folder at the end.

Step 1 — Download the Assignment File from Canvas

  1. Log in to Canvas and go to the assignment you want to complete.
  2. Download the assignment file. It will be a JSON file. The filename will look something like Lab1_Lab_Report_Assignment.json.
  3. The file will save to your Downloads folder. Leave it there for now; you will need it in the next step.

Step 2 — Open GradeBridge and Enter Your Answers

  1. Open a web browser and go to the GradeBridge Student Submission App:
    https://veriqai.github.io/GradeBridge-Student-Submission/

  2. Enter your full name (exactly as it appears in Canvas) and your Student ID.

  3. Upload the assignment JSON file you downloaded in Step 1.

  4. Work through the assignment: Type your answers into the text boxes provided and upload any required screenshots or images into the image fields.

  5. Save your progress before closing the browser.
    Click the Save Backup button. This downloads a backup JSON file to your computer containing everything you have entered so far.

Important: The app does not save automatically. If you close the browser without clicking Save Backup, your work will be lost.

To pick up where you left off another time: return to the app (Step 4), re-enter your name and Student ID (Step 5), and upload your backup JSON file instead of the original assignment file (Step 6).

Step 3 — Download Your Completed Solution Files

  1. When you have finished entering all your answers, click the green Preview and Download button.

  2. Scroll through the preview to check that you have answered every question. If something is missing or incorrect, click Back to Edit (bottom-left of the screen) to return and make corrections.

  3. Once you are satisfied, click the blue Download JSON button. This saves your completed solution as a JSON file — this is the file you will submit to Canvas.

  4. Click the green Download PDF button to save a PDF copy of your submission for your own records. You do not need to turn in the PDF.

Both files will appear in your Downloads folder.

Step 4 — Submit Your Solution on Canvas

  1. Find the solution JSON file in your Downloads folder.

    Tip: Move the file to the folder you set up before starting (e.g., Documents\EEC1\Lab1_Submission\) and rename it to something easy to identify, such as Lab1_Submission_YourLastName.json. Keeping all your submission materials (inputs, backup, and outputs) in one folder makes it easy to find everything later.

  2. Return to Canvas and submit only the solution JSON file.

    Important: Make sure you are uploading the solution file you just generated . Not the original assignment file you downloaded in Step 1. The two files have similar names; double-check before you submit.
