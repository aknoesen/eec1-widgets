Module 7: Applied Op-Amp Circuits

Module 7 puts the Golden Rules to work. Building on the amplifier fundamentals from Module 6, you will extend the op-amp toolkit to include circuits that combine multiple signals and shift DC levels, skills that are essential for the capstone scale project.

This module covers:
* The summing amplifier: adding and scaling multiple input signals simultaneously
* Virtual ground and how the inverting configuration enables the summing operation
* DC level shifting: using a summing amplifier to move a signal into a desired voltage range
* Real-world limits: output saturation and gain-bandwidth product

The Summing Amplifier

The inverting amplifier from Module 6 can be extended to accept multiple input signals at once. Each input contributes to the output independently, scaled by its own resistor ratio. The output is the weighted sum of all inputs, inverted in sign. The key to understanding this circuit is the virtual ground node at the inverting input, which ensures that each input signal is isolated from the others.

DC Level Shifting

A signal that swings between 0 V and 4 V cannot fill an ADC input range of -2.5 V to +2.5 V without adjustment. By treating a fixed reference voltage as one input to the summing amplifier, you can add a precise DC offset to any signal. This level-shifting technique maps a signal from one voltage window to another, and it is a central requirement of the capstone scale design.

Real-World Limits

Two physical limits become important as amplifier gain increases. Saturation occurs when the output tries to exceed the supply rails; the output "clips" and the circuit no longer behaves linearly. The gain-bandwidth product is a fixed property of the op-amp: as gain increases, the usable frequency range decreases. Both limits must be understood before committing to a design.

Hands-On: Lab 7

In Lab 7, you will design and build a summing amplifier that adds a DC offset to shift a signal into a target voltage range.

  • Summing Amplifier: Build an inverting summing amplifier, verify that each input contributes independently to the output, and confirm the expected gain for each channel.
  • Level Shifting: Apply a DC reference to one input and verify that the circuit shifts the signal into the required output range.
